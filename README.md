# au10lucky-notes

## 项目简介

au10lucky-notes 是一个用于归档和发布独立 HTML 页面的仓库。  
这里存放了若干零散的网页文件，它们各自独立，不隶属于任何特定网站或域名。  
本仓库主要用于学习、记录和分享一些静态页面内容。

## 目录说明

```
au10lucky-notes/
├── index.html          # 仓库主页，列出所有归档页面
├── pages/              # 存放各个独立 HTML 页面的目录
│   ├── page1.html
│   ├── page2.html
│   └── ...
└── README.md           # 本文件
```

每个 HTML 页面均保存在 `pages/` 目录下，文件名与内容相对应。  
您可以通过仓库主页或直接访问文件路径查看它们。

## 页面归档说明

- 所有页面均为纯前端实现，不依赖外部后端服务。
- 页面之间无关联，各自独立运行。
- 部分页面可能引用外部资源（如样式库、字体文件等），但不会嵌入跨站追踪或广告。
- 归档内容可能随时间更新或调整，不保证长期不变。
